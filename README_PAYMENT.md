## 🎯 خطوات الإصلاح السريعة

```
1️⃣ افتح: https://dashboard.moyasar.com/settings/api-keys
   ↓
2️⃣ انسخ Secret Key
   ↓
3️⃣ افتح: https://vercel.com/dashboard → Project Settings
   ↓
4️⃣ أضف المتغيرات:
   - MOYASAR_SECRET_KEY = sk_live_...
   - APP_URL = https://bedaih-sa.org
   ↓
5️⃣ أعد النشر (Redeploy)
   ↓
6️⃣ اختبر: افتح الموقع وجرّب الدفع
```
